# Null Hypothesis (H₀) – Easy Explanation

## Clear Definition

**Null Hypothesis (H₀) is a statement that says there is no change, no difference, or nothing new happening.**
It is the **default assumption** that we try to test.

In simple words:
**Null Hypothesis means everything is normal.**

---

## Key Idea to Remember

Null hypothesis always contains:

- Equal $( = )$
- Greater than or equal $( \geq )$
- Less than or equal $( \leq )$

It **never** contains:

- Not equal $( \neq )$
- Greater than $( > )$
- Less than $( < )$

Those belong to Alternative Hypothesis.

---

## Simple Examples

| Situation           | Null Hypothesis (H₀)               |
| ------------------- | ---------------------------------- |
| Teacher             | Average marks $= 70$               |
| Medicine            | Medicine does not work             |
| Machine             | Machine is working properly        |
| New teaching method | New method is not better           |
| Website design      | New design does not increase users |
| Factory             | Packet weight $= 1$ kg             |

See pattern: **No change / No improvement / Same value**

---

## Real-Life Example (Very Important)

### Example: New Teaching Method

A school says new teaching method improves marks.

We test this.

So we assume **opposite first**:

| Hypothesis | Statement                                  |
| ---------- | ------------------------------------------ |
| $H_0$      | New teaching method does NOT improve marks |
| $H_1$      | New teaching method improves marks         |

We always assume **H₀ is true first**, then we test.

---

## Court Example (Best Example)

Think like a court case:

| Concept | Meaning            |
| ------- | ------------------ |
| $H_0$   | Person is innocent |
| $H_1$   | Person is guilty   |

Court assumes **innocent** first (Null Hypothesis).
Then evidence is checked.
If enough evidence → Reject $H_0$ → Person is guilty.

So:
**Null Hypothesis = Default assumption**

---

## Alternative Hypothesis (H₁ / Hₐ)

### 1. What is Alternative Hypothesis?

> **Alternative Hypothesis means: There is change / There is effect / Something new happened.**

It is the **opposite of Null Hypothesis**.

| Symbol | Meaning       |
| ------ | ------------- |
| $H_0$  | No change     |
| $H_1$  | Change exists |

---

### 2. Example (Teaching Method)

| Hypothesis | Statement                         |
| ---------- | --------------------------------- |
| $H_0$      | New method does NOT improve marks |
| $H_1$      | New method improves marks         |

So:

> **H₁ is the claim we actually want to prove.**

---

### 3. Why Do We Need Alternative Hypothesis?

Because we are testing a **claim**.

We cannot test without $H_1$.

Think like this:

| Question                              | Hypothesis |
| ------------------------------------- | ---------- |
| Does medicine work?                   | $H_1$      |
| Does new method improve marks?        | $H_1$      |
| Does new machine increase production? | $H_1$      |

So:

> **Alternative Hypothesis represents the research claim or new idea.**

---

### 4. Real-Life Examples

| Situation   | $H_0$                  | $H_1$             |
| ----------- | ---------------------- | ----------------- |
| Medicine    | Does not work          | Works             |
| Teaching    | No improvement         | Improvement       |
| Machine     | No increase            | Increase          |
| Website     | No increase in users   | Increase in users |
| Study hours | No relation with marks | Relation exists   |

---

### 5. Types of Alternative Hypothesis (Very Important)

There are **3 types**:

| Type         | Symbol           | Example                    |
| ------------ | ---------------- | -------------------------- |
| Right-tailed | $H_1: \mu >$     | New method increases marks |
| Left-tailed  | $H_1: \mu <$     | New method decreases marks |
| Two-tailed   | $H_1: \mu \neq$  | New method changes marks   |

#### Example:

| Case     | $H_0$         | $H_1$              |
| -------- | ------------- | ------------------ |
| Increase | $\mu = 50$    | $\mu > 50$         |
| Decrease | $\mu = 50$    | $\mu < 50$         |
| Change   | $\mu = 50$    | $\mu \neq 50$      |

---

### 6. Visual Understanding

$$H_0: \mu = 50 \quad \text{(no change)}$$

$$H_1: \mu > 50 \quad \text{(increase)}$$

$$H_1: \mu < 50 \quad \text{(decrease)}$$

$$H_1: \mu \neq 50 \quad \text{(change)}$$

---

### 7. Final Summary Table

| Term                        | Meaning                         |
| --------------------------- | ------------------------------- |
| Null Hypothesis $(H_0)$     | No change                       |
| Alternative Hypothesis $(H_1)$ | Change exists                |
| Why $H_1$                   | Because we want to test a claim |
| Goal                        | Reject $H_0$ to support $H_1$   |

---

### 8. One-Line Summary (Remember This)

> **Alternative Hypothesis is the claim we want to prove, but we prove it by rejecting the Null Hypothesis.**

---

### 9. Full Logic in One Flow

```
We want to prove H₁
But we assume H₀ is true
We collect data
If data gives strong evidence
We reject H₀
So we support H₁
```

## Final One Line Definition

**Null Hypothesis is the default statement that says there is no difference or no effect, and we try to test whether it is true or false.**

---

## Null Hypothesis and Alternative Hypothesis

---

## 1. Basic Idea

In statistics, we always make **two statements**:

| Symbol | Name                      | Meaning                |
| ------ | ------------------------- | ---------------------- |
| $H_0$  | Null Hypothesis           | No change / No effect  |
| $H_1$  | Alternative Hypothesis    | Change / Effect exists |

> We assume **$H_0$ is true first**, then we try to **reject it** using data.
> If we reject $H_0$ → we support $H_1$.

---

## 2. Simple Example – Average Marks

Suppose a teacher says:

> "Average marks of students is 70."

This is a **claim about population mean** $(\mu)$.

Now we write hypotheses:

| Hypothesis | Statement       |
| ---------- | --------------- |
| $H_0$      | $\mu = 70$      |
| $H_1$      | $\mu \neq 70$   |

This means we are checking whether the average is **different from 70 or not**.

This is called a **two-tailed test**.

---

## 3. Another Example – New Teaching Method

Claim:

> New teaching method improves marks.

We convert into hypotheses:

| Hypothesis | Statement                         |
| ---------- | --------------------------------- |
| $H_0$      | New method does NOT improve marks |
| $H_1$      | New method improves marks         |

In mean form:

| Hypothesis | Mathematical Form     |
| ---------- | --------------------- |
| $H_0$      | $\mu_1 = \mu_2$       |
| $H_1$      | $\mu_1 > \mu_2$       |

Where:

- $\mu_1$ = mean marks (new method)
- $\mu_2$ = mean marks (old method)

---

## 4. Important Rule (Very Important for Exams & Interviews)

| Rule                              | Explanation                      |
| --------------------------------- | -------------------------------- |
| Equality $(=)$ always in $H_0$    | Because $H_0$ is default assumption |
| $H_1$ contains $>$, $<$, or $\neq$ | Because $H_1$ shows change      |

---

## 5. Full Logic Flow

```
Step 1: Write H₀ (No change)
Step 2: Write H₁ (Change exists)
Step 3: Collect sample data
Step 4: Perform statistical test
Step 5: Calculate p-value
Step 6:
    If p-value < 0.05 → Reject H₀ → Support H₁
    If p-value ≥ 0.05 → Fail to Reject H₀
```

---

## 6. Decision Table

| $p$-value    | Decision             | Conclusion                   |
| ------------ | -------------------- | ---------------------------- |
| $p < 0.05$   | Reject $H_0$         | Evidence for $H_1$           |
| $p \geq 0.05$ | Fail to Reject $H_0$ | Not enough evidence for $H_1$ |

---

## 7. One Real-Life Example (Complete)

**Problem:** A company says average salary $= 30{,}000$.

| Hypothesis | Statement          |
| ---------- | ------------------ |
| $H_0$      | $\mu = 30{,}000$   |
| $H_1$      | $\mu \neq 30{,}000$ |

We collect sample salary data and perform test:

- If $p\text{-value} = 0.02$ → Reject $H_0$ → Salary is different from $30{,}000$
- If $p\text{-value} = 0.30$ → Fail to Reject $H_0$ → No proof that salary is different

---

## 8. Final Summary (Remember This)

| Concept                   | Meaning        |
| ------------------------- | -------------- |
| Null Hypothesis           | No change      |
| Alternative Hypothesis    | Change         |
| We assume                 | $H_0$ is true  |
| We try to                 | Reject $H_0$   |
| If rejected               | Support $H_1$  |
| If not rejected           | Keep $H_0$     |

---

## Final One-Line Concept

> **Null Hypothesis = No change**
> **Alternative Hypothesis = Change**
> **We use data to reject Null Hypothesis and support Alternative Hypothesis.**
