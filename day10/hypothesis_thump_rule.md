# Thumb Rule to Write H₀ and H₁

## Rule 1

> **Null Hypothesis (H₀) always contains equality**
> (= or ≤ or ≥)

## Rule 2

> **Alternative Hypothesis (H₁) contains inequality**
> (> or < or ≠)

## Rule 3

> **Alternative Hypothesis (H₁) is usually the claim we want to prove**

---

# Thumb Rule Table

| Words in Question | H₀  | H₁  |
| ----------------- | --- | --- |
| Equal to          | =   | ≠   |
| Not equal to      | =   | ≠   |
| Greater than      | ≤   | >   |
| Less than         | ≥   | <   |
| At least          | ≥   | <   |
| At most           | ≤   | >   |
| Increase          | ≤   | >   |
| Decrease          | ≥   | <   |
| Better / Improve  | ≤   | >   |
| Worse             | ≥   | <   |
| Different         | =   | ≠   |

---

# Step-by-Step Thumb Rule

When a question comes, do this:

```id="rulebox1"
Step 1: Convert statement into math (μ >, μ <, μ =)
Step 2: Put inequality sign in H₁
Step 3: Put equality sign in H₀
Step 4: H₀ and H₁ must be opposite
```

---

# Example Using Thumb Rule

## Example 1

Statement: Average marks > 70

| H₀ | μ ≤ 70 |
| H₁ | μ > 70 |

---

## Example 2

Statement: Average marks = 70

| H₀ | μ = 70 |
| H₁ | μ ≠ 70 |

---

## Example 3

Statement: New method improves marks

“Improves” = increase = >

| H₀ | μ_new ≤ μ_old |
| H₁ | μ_new > μ_old |

---

## Example 4

Statement: Medicine reduces BP

“Reduces” = decrease = <

| H₀ | μ ≥ 120 |
| H₁ | μ < 120 |

---

# One-Line Thumb Rule (Remember This)

> **Inequality goes to H₁, Equality goes to H₀**

# Thumb Rule – Tail Identification

## Ultimate Rule

> **Tail is decided by Alternative Hypothesis (H₁), not H₀.**
> Look at **H₁ symbol**:

| H₁ Symbol | Tail Type  |
| --------- | ---------- |
| >         | Right Tail |
| <         | Left Tail  |
| ≠         | Two Tail   |

---

# Correct Word → Tail Rule

| Words in Question                                    | Convert to Math | Goes to | H₁ becomes | Tail  |
| ---------------------------------------------------- | --------------- | ------- | ---------- | ----- |
| Increase / Improve / Greater / More / Higher / Above | >               | H₁      | >          | Right |
| Decrease / Reduce / Less / Lower / Below             | <               | H₁      | <          | Left  |
| Different / Not equal / Change                       | ≠               | H₁      | ≠          | Two   |
| Equal / Average / Mean                               | =               | H₀      | ≠          | Two   |
| At least                                             | ≥               | H₀      | <          | Left  |
| At most                                              | ≤               | H₀      | >          | Right |

**This is the corrected table.**

| --------- | ----- |

# Super Simple Memory Trick

Remember this sentence:

> **“More → Right, Less → Left, Different → Two”**
> **“At least → Left, At most → Right”**

---

# 4-Step Method (Always Correct)

1. Convert sentence to symbol (>, <, =, ≥, ≤)
2. Write that as **H₀**
3. Write opposite as **H₁**
4. Tail depends on **H₁ symbol**

---

# Example

**Average profit is at most 50,000**

| Step     | Result     |
| -------- | ---------- |
| Sentence | At most    |
| Symbol   | ≤          |
| H₀       | μ ≤ 50,000 |
| H₁       | μ > 50,000 |
| Tail     | Right      |
