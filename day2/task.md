Day 1 Excel Practice Tasks (Basics + Conditional Formatting)

Task 1 – Student Fees Table
Create a table with columns: Student Name, Course, Fees, Paid, Balance.
Enter at least 5 students.
Calculate Balance = Fees – Paid.
Apply borders, bold headings, header fill color, and center alignment.

Task 2 – Highlight Balance
Highlight students who have not paid full fees.
Apply conditional formatting on Balance column where Balance > 0.

Task 3 – Highlight Python Students
Highlight rows where Course = Python using conditional formatting.

Task 4 – Highlight Highest Fees
Highlight the highest value in the Fees column using Top/Bottom rule.

Task 5 – Duplicate Values
Create a Product column and enter some duplicate product names.
Highlight duplicate values using conditional formatting.

Task 6 – Sales Table
Create columns: Date, Sales Person, Region, Product, Sales Amount.
Apply conditional formatting:
Sales > 2000 → Green
Region = East → Blue
Product = Mouse → Yellow

Task 7 – Attendance
Create columns: Name, Attendance %.
Highlight attendance less than 75%.

Task 8 – Alternate Row Color
Create any table and highlight alternate rows using conditional formatting.

Task 9 – Expiry Date
Create columns: Product, Expiry Date.
If expiry date is less than today → Red.
If expiry date is within next 5 days → Yellow.

Task 10 – Salary Sheet
Create columns: Employee, Salary.
Salary > 30000 → Green.
Salary < 20000 → Red.

Task 11 – Yes/No Status
Create columns: Name, Assignment Submitted (Yes/No).
Highlight No → Red.
Highlight Yes → Green.

Task 12 – Marks Sheet
Create columns: Student, Marks.
Marks ≥ 80 → Green.
Marks 50–79 → Yellow.
Marks < 50 → Red.

Task 13 – Highlight Entire Row
Create columns: Name, Department, Salary.
Highlight entire row where Department = IT.

Task 14 – Duplicate IDs
Create a column ID with some duplicate numbers.
Highlight duplicate IDs and unique IDs.

Task 15 – Product Bill
Create columns: Product, Price, Quantity, Total.
Calculate Total = Price × Quantity.
Highlight Total > 5000 → Green.
Highlight Total < 1000 → Red.

Task 16 – Student Result
Create columns: Name, Subject, Marks, Result.
If Marks < 35 → Fail (Red).
If Marks ≥ 35 → Pass (Green).

Task 17 – Top Performers
Create a Marks column for 10 students.
Highlight top 3 marks and highlight entire row.

Task 18 – Data Formatting
Create any table and do the following:
Bold headings
Apply borders
Adjust column width
Center align text
Format numbers as currency

Task 19 – Highlight Specific Text
Create a table with Region column.
Highlight cells where Region = Americas.

Task 20 – Mini Project
Create a Mobile Shop Bill with Item, Price, Quantity, Total.
Calculate total and apply conditional formatting based on total amount.
